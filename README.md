# EchoScape.ai

EchoScape.ai is tool that provide Sppech to Image facility. It used openai Whisper and Dalle for this task

### How To Run The App
- Clone the project
- Open the project cmd and run npm install
- run npm start
- go to localhost:3000 to see the app
